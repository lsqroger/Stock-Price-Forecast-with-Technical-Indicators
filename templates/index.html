Forecast Stock Price Change with Technical Indicators and Machine Learning Model

A mini project by Roger Liu

This project delivers a ML predictive model that predicts the direction of price change for a specified stock. The model is utilizing various technical indicator (TA) values of the stock at the moment to forecast the stock's price change 7 days later (i.e. 'Up' or 'Down'), with recent 2 years of historical daily price and TA data used as training set. Historical data of stocks are extracted on demand from Yahoo!Finance website through yfinance python package, and technical indicators are constructed using ta python package.
  • Model: LightGBM
  • Feature: Technical Indicators
  • Feature Selection: Embedded
  • Hyperparameter Tuning: Bayesian
